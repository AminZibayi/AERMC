# Citation Standards & Reference Rules

## 16. Reference Standards

1. **Verbatim Citations**: All extracted quotes must use exact-text citation format:

   > exact phrase "..." from `Source` (Section, Item, or Note reference)

2. **Page Numbers**: If pagination is unavailable, only reference the Item or Note number. Do not fabricate page numbers.

3. **Source Priority**: 10-K > Proxy > 8-K > Yahoo Finance > Tavily (for ERM content).

4. **Framework Citations**: When naming frameworks (COSO, ISO 31000, Basel III, NIST), cite the exact wording found in the filing, even if paraphrased labels are used.

5. **Reference Numbering**: Maintain a consistent numbering system for all VERBATIM citations (e.g., [1], [2], [3]) in the final report's Reference section.

## 17. Prohibited Practices

1. **No Fabrication**: Do not quote content not present in the raw SEC filing text or Yahoo Finance data.

2. **No Paraphrase Without Source**: Any statement summarizing risk content must be traceable to a verbatim quote in the report.

3. **No SEC/SEC.gov URLs**: Do not generate, guess, or embed any SEC.gov URLs — reference only by accession number and form type.

4. **No Secret Logging**: Do not include API keys, credentials, or environment variables in the report output.

5. **No Out-of-Scope Recommendations**: Do not recommend specific investment actions. Limit analysis to risk assessment per the 13-phase protocol.

6. **No Inferred Numbers Without Label**: Any derived ratio or metric must be labeled "Derived" in the Source column of output tables.

7. **No Semantic Leakage**: Do not infer facts not supported by the source text. If information is absent, state "Not disclosed in filing."
